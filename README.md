# iFeed-for-iOS
A Simple and beatiful rss app that fetch rss feeds from websites

## Installation ##
* Git and clone <br/>
```bash
$ cd /Users/user/my_project
$ git clone https://github.com/TarokhDev2020/iFeed-for-iOS.git
$ pod install
```

## Design Pattern ##
* MVC
